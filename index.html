Dannii's Interactive Fiction site

I am a fan of interactive fiction (IF) and am involved with a number of things which I hope will help the community and push the medium forward.

Projects

I maintain several interactive fiction software projects, all of which are hosted at GitHub:

  • Gnusto: a Javascript Z-Machine interpreter
  • Parchment: a web interpreter for IF, currently including Gnusto, but soon to have Glulx support as well
  • Parchment for Inform 7: the template used for producing a custom Parchment site in Inform 7
  • Parchment-proxy: a simple file proxy used by Parchment which run's on Google's App Engine

All of the above projects use the old Parchment Google Code bug tracker.

Standards

Recently I have begun working on several new technical standards:

  • Glulx extensions
  • The Z-Machine Standard 1.2 (draft)
